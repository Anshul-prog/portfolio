This folder is for your demo video MP4 files.

To make the demo sections work out-of-the-box, simply place your MP4 files here and name them exactly as follows:

1. `chatbot-demo.mp4` -> For the first demo section (AI Website Chatbot / Inbound Lead Qualification Flow)
2. `calling-bot-demo.mp4` -> For the second demo section (Voice AI Sales Representative)
3. `lead-qualifier-demo.mp4` -> For the third demo section (Automated Omni-Channel Support)

If you use different filenames, make sure to update the `src` attributes in the <video> tags inside `app/page.tsx`!
